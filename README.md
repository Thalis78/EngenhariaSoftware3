## Descrição do Projeto:
O objetivo deste trabalho foi testar um sistema de **Achados e Perdidos**, que permite o cadastro e a visualização de itens perdidos e encontrados. O foco principal foi realizar **testes de desempenho** para avaliar a robustez e o comportamento do sistema sob carga, utilizando duas ferramentas principais: **JMeter** para o teste de carga e **Selenium** para a automação de testes funcionais.

- **JMeter** foi utilizado para simular múltiplos usuários acessando o sistema simultaneamente, visando analisar o desempenho do sistema sob diferentes cargas de uso.
- **Selenium** foi utilizado para a automação de testes funcionais do sistema, garantindo que as funcionalidades principais estivessem funcionando corretamente de forma automatizada, como a inserção, visualização e exclusão de itens.

## Ferramentas e Linguagens Utilizadas:
- **Linguagem**: Java
- **Framework de Automação**: Selenium WebDriver
- **Teste de Desempenho**: Apache JMeter
- **Outras Ferramentas**: Maven (para gerenciamento de dependências)
